Політика конфіденційності

Тут коротко пояснено, які дані ми можемо отримувати через сайт DsPrint Наклейки, навіщо вони потрібні та як використовуються під час обробки звернень і замовлень.

1. Які дані ми отримуємо

Тільки те, що ви самі надсилаєте

Ми можемо отримувати ім’я, номер телефону, email, посилання на макет, текст коментаря, дані для доставки та іншу інформацію, яку ви добровільно залишаєте у формах сайту або надсилаєте через Telegram, Viber, email чи телефонний зв’язок.

2. Для чого використовуються дані

Щоб відповісти й виконати замовлення

Контактна інформація використовується для зв’язку з вами, уточнення параметрів замовлення, перевірки макета, погодження ціни, способу доставки, термінів виготовлення та подальшого супроводу заявки.

3. Форми на сайті

Дані з заявок не публікуються

Інформація, яку ви залишаєте через калькулятор, форму замовлення або форму зворотного зв’язку, використовується лише для внутрішньої обробки звернення. Ми не публікуємо такі дані у відкритому доступі.

4. Передача третім сторонам

Лише коли це потрібно для замовлення

Ваші дані можуть використовуватися лише в обсязі, необхідному для виконання замовлення, наприклад для доставки або зв’язку. Ми не продаємо та не передаємо ваші контактні дані стороннім особам без потреби, пов’язаної з обробкою замовлення.

5. Telegram, Viber, телефон

Комунікація поза формами

Якщо ви звертаєтесь до нас через Telegram, Viber або телефон, ми можемо використовувати надані вами контактні дані та зміст звернення для подальшого супроводу замовлення, відповіді на запитання та погодження деталей друку.

6. Аналітика та технічні дані

Можливі базові технічні журнали

Під час роботи сайту можуть автоматично фіксуватися стандартні технічні дані, пов’язані з відкриттям сторінок, роботою браузера або завантаженням контенту. Такі дані використовуються для стабільної роботи сайту та не призначені для публічного розкриття.

7. Зберігання даних

Стільки, скільки потрібно для роботи

Контактні дані та листування можуть зберігатися протягом часу, необхідного для комунікації, виконання замовлення, повторного звернення клієнта або уточнення попередніх домовленостей.

8. Оновлення політики

Можливі зміни

Ми можемо оновлювати цю політику, якщо змінюється логіка форм, канали комунікації або порядок обробки замовлень. Актуальна версія завжди публікується на цій сторінці.

Контакти

Потрібно уточнення щодо ваших даних?
